Manual & Workflow
e-Panta User Guide
Cloud Backup: మీ డేటా క్లౌడ్‌లో సురక్షితంగా బ్యాకప్ అవుతుంది — మీరు వేరే డివైస్‌లో సైన్ ఇన్ చేసినా ఫైల్స్ మళ్ళీ అప్‌లోడ్ చేయాల్సిన అవసరం లేదు.
1 పంట నమోదు వివరాలు డౌన్లోడ్

ముందుగా ఈ పంట నుంచి పంట నమోదు వివరాలు డౌన్లోడ్ చేసుకోవాలి. Village Profile కొరకు పంట నమోదు వివరాలు MAO లాగిన్ నుంచి Pattadhar/Cultivator ఉన్న ఫైల్ ని డౌన్లోడ్ చేసుకోండి.

2 Excel ఫైల్ కాలమ్ పేర్లు మార్చడం

ఎక్సెల్ ఫైల్ లో క్రింద చూపిన విధంగా కాలమ్ పేర్లు మార్చుకోగలరు:

Required Column Names
Booking ID
Khatha Number
Survey Number
Area
పైన తెలిపిన విధంగా కాలమ్ పేర్లు మార్చనియెడల రైతుల వివరాలు మరియు పంట వివరాలు రావు.
3 Upload Report

Upload Report లో కాలమ్ పేర్లు మార్చిన Excel ఫైల్ ని అప్లోడ్ చేయాలి.

4 e-Panta Report

అప్లోడ్ చేసిన తరువాత e-Panta Report లో పంటల వారీగా రైతుల సంఖ్య, విస్తీర్ణం మరియు రైతుల పేర్లు రావడం జరుగుతుంది.

5 Search Farmer

ఒక రైతు యొక్క పంట నమోదు వివరాలు చూడాలి అనుకుంటే Dashboard లో Search Farmer దగ్గర ఆధార్ కార్డు చివరి నాలుగు అంకెలు ద్వారా చూడవచ్చు.

గమనిక: ఒకవేళ ఇద్దరు రైతులకి ఆధార్ కార్డు చివరి నాలుగు అంకెలు ఒకేలా ఉంటే, ఇద్దరివి కలిపి ఒకరికే చూపెడుతుంది. అలా ఉన్నప్పుడు ఖాతా నెంబర్ ను పరిశీలించగలరు.
